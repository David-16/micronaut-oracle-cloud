2 Quick Start

To get started quickly Setup the Oracle Cloud CLI making sure you have run oci setup config to configure local access to Oracle Cloud.

Then add a dependency on the micronaut-oci-sdk module:

implementation("io.micronaut.oci:micronaut-oci-sdk")
<dependency>
    <groupId>io.micronaut.oci</groupId>
    <artifactId>micronaut-oci-sdk</artifactId>
</dependency>

Followed by a dependency on the OCI SDK client you wish to use, for example for Object Store:

implementation("io.micronaut:com.oracle.oci.sdk:oci-java-sdk-objectstorage")
<dependency>
    <groupId>io.micronaut</groupId>
    <artifactId>com.oracle.oci.sdk</artifactId>
    <version>oci-java-sdk-objectstorage</version>
</dependency>

All OCI SDK clients are supported.
